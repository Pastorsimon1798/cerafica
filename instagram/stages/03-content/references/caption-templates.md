# Caption Templates

Starting points for different pillar types. Adapt to your voice.

## Finished Work (Sales Posts)

### Template A: Available Now
```
[Piece name/description] is looking for a home.

[2-3 sentences about the piece -- size, texture, what makes it special]

[Dimensions or details if relevant]

DM to claim. [Price if sharing]
```

### Template B: Story + Available
```
[Story about making this piece -- what inspired it, how it came to be]

That's why [piece name] feels [emotion/quality].

Available now. DM to claim.
```

### Template C: Collection Drop
```
Fresh from the kiln.

[Number] new pieces, each one [unique quality].

[Swipe/See] through for details. DM to claim your favorite.
```
