# Task Routing

| I want to... | Go to |
|--------------|-------|
| Set up my brand identity for the first time | `setup/questionnaire.md` |
| Add new photos/videos for this week's content | `instagram/stages/01-input/CONTEXT.md` |
| Decide what to post this week | `instagram/stages/02-planning/CONTEXT.md` |
| Write captions for my posts | `instagram/stages/03-content/CONTEXT.md` |
| Create stories or reels from my content | `instagram/stages/04-repurposing/CONTEXT.md` |
| Generate and post content | `./instagram/post` |
| Review or update my brand identity | `brand/identity.md` |
| Check my voice guidelines | `brand/voice-rules.md` |
| Find hashtag sets to use | `shared/hashtag-library.md` |
| Update product inventory | `inventory/products.json` |
| Edit my website | `website/` |

## Quick Start

New week? Start here:
1. `instagram/stages/01-input/CONTEXT.md` -- collect your assets
2. Follow the pipeline through stages 02-04
3. Post with `./instagram/post`

First time? Run `setup` to configure your brand identity.
