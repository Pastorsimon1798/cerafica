# Brand Vault

Persistent brand identity files that inform all content creation.

## Files

| File | Purpose |
|------|---------|
| `identity.md` | Who you are, aesthetic, goals, positioning |
| `voice-rules.md` | How you sound -- do's, don'ts, examples |

## When to Update

- Quarterly brand review
- After completing the setup questionnaire
- When your aesthetic or goals shift
- When voice feels off in generated captions

## How These Files Work

These files are **inputs** to content creation:
- `instagram/stages/02-planning` reads identity for goal alignment
- `instagram/stages/03-content` reads voice-rules for caption generation
- All stages reference these for consistency

## Editing

These files are designed to be edited directly. Unlike stage outputs (regenerated weekly), brand vault files persist and evolve slowly.
