## Global Reach Strategy (Beyond Long Beach)

### Current State
- Etsy shop: cerafica.etsy.com
- No own website
- No email list

### The Hybrid Approach (Recommended)

```
Instagram → Drive traffic to BOTH
    ├── Etsy (discovery, built-in buyers)
    └── Your own website (brand building, email list, higher margins)
```

### Etsy vs Own Website (2026)

| Factor | Etsy | Own Website |
|--------|------|-------------|
| Setup | Easy, free | Requires work, $10-30/mo |
| Built-in traffic | 93.9M buyers | None - you drive it |
| Fees | 6.5% + listing + payment | ~3% payment processing |
| Customer data | Limited | Full access (email!) |
| Trust | High | Build it yourself |
| Risk | Can be suspended | You control it |

### International Shipping (For Global Reach)

| Element | Recommendation |
|---------|----------------|
| **Markets** | Canada, UK, Australia, EU |
| **Shipping** | Flat rate for small, calculated for large |
| **Packaging** | Double-box, lots of padding |
| **Customs** | Mark as "handmade ceramic art" |
| **Pricing** | Build shipping into price for "free shipping over $X" |

### Email List (The Missing Piece)

You don't own your Instagram followers. You DO own your email list.

**Lead magnets for potters:**
- "First access to new pieces"
- "Workshop notifications"
- "Behind-the-scenes studio updates"
- "Subscriber-only discounts"
