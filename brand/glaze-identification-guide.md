# Glaze Identification Guide

> Reference for visual glaze identification. Imported from GlazeLab systematic approach.

---

## CRITICAL: Mandatory Process

**NEVER identify a glaze family without completing the systematic comparison.**

The #1 error in glaze identification is anchoring to secondary features (metallic sheen, texture) while ignoring the PRIMARY diagnostic: **base color**.

---

## Stage 1: Observation Protocol

### Required Data Collection

Before any identification, document:

| Attribute | What to Look For |
|-----------|------------------|
| **Primary base color** | The dominant color across the piece |
| **Undertones** | Green? Brown? Blue? Red? (CRITICAL) |
| **Surface gloss** | High gloss / satin / matte |
| **Translucency** | Translucent / opaque / semi-opaque |
| **Pooling behavior** | Darker where thick? Lighter? No pooling? |
| **Color transitions** | Gradual / sharp / breaking at edges |
| **Metallic effects** | Present / absent / where? |
| **Texture** | Smooth / crawling / crackle / crystalline |
| **Clay body visibility** | Color where exposed |

---

## Stage 2: Decision Tree

```
START: What is the PRIMARY base color?

├─ GREEN or BLUE-GREEN
│   ├─ Translucent with glossy pooling? → CELADON
│   ├─ Light blue with purple flash? → CHUN BLUE
│   ├─ Copper green with red flashes? → ORIBE
│   └─ Variegated, earthy? → ASH GLAZE
│
├─ BROWN or BLACK
│   ├─ Color breaking to rust at edges? → TENMOKU
│   ├─ Heavy texture, rust surface? → IRON SATURATE
│   └─ Variegated, organic? → ASH GLAZE
│
├─ WHITE or CREAM
│   ├─ Crawling texture? → SHINO
│   ├─ Opaque white? → TIN GLAZE
│   └─ Satin/matte? → MATTE WHITE
│
├─ ORANGE or TAN
│   ├─ With crawling? → SHINO
│   └─ With carbon trap speckles? → CARBON TRAP SHINO
│
└─ OTHER
    └─ Check colorant-specific effects
```

## Critical Distinction: Celadon vs Tenmoku

**This is the #1 confusion point. Get it right.**

| Feature | Celadon | Tenmoku |
|---------|---------|---------|
| **Primary base color** | GREEN/blue-green | BROWN/black |
| **Iron content** | LOW (0.5-2%) | HIGH (6-10%) |
| **Translucency** | Translucent | Less translucent |
| **Pooling** | Darker green where thick | Darker brown/black where thick |
| **Key signal** | **GREEN undertones** | **BROWN undertones** |
| **Color breaking** | No | Yes (rust at edges) |

**Decision Rule:**
- Base color has **GREEN** undertones → **CELADON**
- Base color is **BROWN/BLACK** with rust breaking → **TENMOKU**

---

## Critical Distinction: Shino vs Teadust

**For our studio specifically.**

| Feature | Shino | Teadust |
|---------|-------|---------|
| **Surface** | TEXTURED (crackle, crawling, variegation) | SMOOTH uniform speckle |
| **Appearance** | Organic, broken, irregular | Uniform golden-green speckle |
| **Crawling** | Common | Never |
| **Crackle** | Common | Never |

**Decision Rule:**
- Any crackle, crawling, or texture → **SHINO**
- Smooth, uniform golden speckle → **TEADUST**

## Colorant Behavior Reference

### Iron Oxide (Fe₂O₃)

| Concentration | Oxidation | Reduction |
|---------------|-----------|-----------|
| 0.5-2% (LOW) | Tan, cream | **GREEN** (celadon) |
| 2-4% | Light brown | Green-brown |
| 4-6% | Brown | Dark brown |
| 6-10% (HIGH) | Rust, brick red | **BLACK/BROWN** (tenmoku) |
| 10%+ | Red-brown | Dark brown, runs |

**Key insight:** Iron in reduction produces GREEN at low concentration, BROWN/BLACK at high concentration.

### Copper Oxide (CuO)

| Concentration | Oxidation | Reduction |
|---------------|-----------|-----------|
| 0.5-2% | Green/turquoise | **RED/metallic copper** |
| 2-5% | Bright green | Red with metallic |

**Key insight:** Metallic copper (Cu⁰) = CONFIRMS reduction firing

### Cobalt Oxide (CoO)

| Concentration | Effect |
|---------------|--------|
| 0.5-3% | Deep blue (both atmospheres) |

## Firing Atmosphere Indicators

### Reduction Firing (oxygen-starved)

| Indicator | What You See |
|-----------|--------------|
| Green undertones (iron) | Fe²⁺ = green |
| Metallic copper | Cu⁰ = metallic gold/red |
| Dark browns/blacks | Fe²⁺ concentrated |
| Color breaking at edges | Iron saturation effect |

### Oxidation Firing (oxygen-rich)

| Indicator | What You See |
|-----------|--------------|
| Bright, stable greens | CuO unchanged |
| Rust/tan/bright browns | Fe₂O₃ unchanged |
| No metallic effects | Metals stay oxidized |

## Anti-Patterns (DO NOT DO THIS)

### ❌ Anti-Pattern 1: Anchoring to Metallic Sheen

**Wrong:** "It has metallic sheen, must be Tenmoku."
**Right:** Metallic sheen occurs in many glazes. Use base color as primary signal.

### ❌ Anti-Pattern 2: Confusing Pooling with Color Breaking

**Wrong:** "It's darker at the bottom, that's color breaking."
**Right:**
- Pooling = darker where THICK (gravity, recesses)
- Color breaking = darker at EDGES (thin areas, application)

### ❌ Anti-Pattern 3: Over-weighting Secondary Features

**Wrong:** "The metallic sheen and texture mean it's Tenmoku."
**Right:** Base color is 70% of the signal. Start there.

## Surface Effects Reference

### Intentional Effects (Feature, Not Bug)

| Effect | Visual | Cause |
|--------|--------|-------|
| **Color breaking** | Different color at edges | Iron saturation, thin areas |
| **Pooling** | Darker in recesses | Translucent glaze behavior |
| **Variegation** | Mottled, textured | Variable thickness, colorants |
| **Speckling** | Speckled surface | Iron, rutile, ilmenite |
| **Carbon trapping** | Black speckles | CO₂ trapped (shino) |
| **Oil spotting** | Metallic sheen | Oil spot technique |
| **Tea dust** | Green/gold speckles | Tea dust spray |
| **Crystals** | Visible crystals | Slow cooling, zinc/titanium |
| **Luster** | Metallic sheen | Metallic colorants, reduction cool |
| **Crawling** | Exposed clay | Intentional texture (shino) |

## Confidence Assessment

After identification, rate confidence:

| Component | Rate Based On |
|-----------|---------------|
| Glaze family | Match to decision tree + observation strength |
| Firing atmosphere | # of consistent indicators |
| Colorant | Match to expected behavior |

### Confidence Levels

- **HIGH**: 3+ consistent indicators, matches decision tree exactly
- **MEDIUM**: 2 consistent indicators, mostly matches
- **LOW**: 1 indicator, ambiguous, or conflicts present
