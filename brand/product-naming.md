# Cerafica Product Naming Conventions

> How every vessel gets its planetary designation.
> Version: 1.0.0

---

## 1. Naming Philosophy

Every Cerafica piece is a **world**.

The name should sound like:
- An exoplanet discovered by a distant observatory
- A mineral sample catalogued from an asteroid belt
- A sector on a sci-fi star chart

**Not like:**
- A furniture catalog item
- A yoga studio class name
- A candle scent

#3 3. Root Construction

The root is typically **2-4 syllables** and evokes:
- A chemical element
- A mineral or rock type
- A color
- A geological feature
- A firing or glaze characteristic

##3 Root Sources

| Category | Examples |
|----------|----------|
| **Elements** | Pall (palladium), Crom (chromium), Cupr (copper), Pyr (pyrite/fire) |
| **Minerals** | Lazur (lapis lazuli), Mari (marble), Bitum (bitumen) |
| **Colors** | Cerule (cerulean), Glac (glacial blue), Ign (ignition orange/red) |
| **Geology** | Krel (crystalline), Nex (nexus/convergence) |
| **Process** | Carb (carbon), Tenm (tenmoku), Shino (shino) — use sparingly |

---

## 4. Suffixes

The suffix adds the **planetary/scientific flavor**.

| Suffix | Vibe | Examples |
|--------|------|----------|
| `-ix` | Standard exoplanet | Cromix, Ignix, Glacix |
| `-ex` | Mineral compound | Krel-ex, Cupr-ex |
| `-os` | Greek/planetary | Pyr-os, Bitum-os |
| `-un` | Latin/ancient | Nex-un |
| `-th` | Ordinal/technical | Pallth |
| `-rix` | Grand/celestial | Lazurix, Ceruleix |
| `-on` | Particle/physics | *Available for future use* |
| `-ar` | Stellar | *Available for future use* |

##3 Suffix Selection Rules

1. **Default to `-ix`** for most pieces. It's the standard "exoplanet" sound.
2. **Use `-ex`** when the root ends in a hard consonant (Krel-ex, Cupr-ex).
3. **Use `-os`** when the root evokes fire, earth, or darkness (Pyr-os, Bitum-os).
4. **Use `-un`** sparingly. It sounds ancient and mysterious.
5. **Use `-th` and `-rix`** for statement pieces or flagship forms.

---

## 5. The Number

- Always a **single digit** (0-9)
- The number indicates the **iteration** of that name in the catalog
- `Cromix-0` = the first Cromix
- `Pallth-7` = the eighth iteration of Pallth

**Rules:**
- If a name has never been used before, start at `-0`
- If reusing a root+suffix combo, increment the number
- Do not skip numbers
- Do not use double digits

#3 7. Product Descriptions

Each description should read like a **planetary survey report** from orbit.

##3 Description Formula

```
[Geological feature] + [light behavior] + [material chemistry] + [ancient history]
```

##3 Themes to Draw From
- **Geology:** fracture networks, tectonic basins, craters, fault lines, ridges
- **Light:** catches the light, pools in glossy memories, false dusk, metallic sheen
- **Chemistry:** iron-manganese, cobalt-iron, carbon trap shino, cone 10 reduction
- **Time:** ancient bombardment, geological time, orbital decay, formation epochs

### Example Descriptions

> "Fracture networks catch the light — each line a fault from an ancient bombardment still exhaling the planet's history."

> "A world of impossible chun crystallization and metallic luster. The false dusk is cast by cobalt-iron skies."

> "Rare chun crystallization pools in tectonic basins. The surface shifts with geological history written in impossible blues and seafoam bands."

> "Iron-manganese deposits create a variegated terrain. The glaze broke at the rim, revealing rust-colored strata beneath."

#3 9. Naming Checklist

Before finalizing a name:

- [ ] Does it sound like a planet or mineral sample?
- [ ] Is the root 2-4 syllables?
- [ ] Does the suffix match the root's ending sound?
- [ ] Is the number a single digit (0-9)?
- [ ] Has this exact root+suffix combo been used before?
- [ ] Does the description include geological + light + chemistry language?
- [ ] Is the piece framed as "one of one"?
